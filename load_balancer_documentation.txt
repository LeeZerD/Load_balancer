Etape 1 : 
lancement instances serveurs web: 
~/Documents/Rust/servers$ docker-compose up
(bloque le terminal pour visualiser les logs)

Etape 2 : 
lancement du load balancer : 
~/Documents/Rust/load_balancer$ cargo run

Etape 3 : 
curl http://localhost:8080 (ne devrait rien renvoyer si les deux étapes précédentes ne sont pas validées)

Etape 4 : 
test de la connectivité : 
wrk -t12 -c400 -d30s http://localhost:8080
Options:                                            
    -c, --connections <N>  Connections to keep open   
    -d, --duration    <T>  Duration of test           
    -t, --threads     <N>  Number of threads to use
